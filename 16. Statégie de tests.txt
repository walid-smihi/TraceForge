# Stratégie de tests — TraceForge

---

## Objectifs de couverture

Backend : > 80% de couverture sur les services et routers
Frontend : > 60% sur les composants critiques (formulaires, tableau exigences, graphe)
Pas de test E2E obligatoire avant le Sprint 7

---

## Pyramide des tests

Unitaires (rapides, beaucoup) → Services, parsers, règles métier, LLM providers
Intégration (moyen, quelques-uns) → Routes FastAPI avec vraie DB de test
E2E (lents, peu) → Scénario complet (Sprint 7 avec Playwright)

---

## Tests backend (pytest)

### Structure

```
apps/api/tests/
  conftest.py              # fixtures globales (DB, client HTTP, mock LLM)
  test_projects.py         # CRUD projets
  test_documents.py        # upload, extraction texte
  test_requirements.py     # extraction, classification, ambiguïté
  test_trace_links.py      # suggestions, validation, règles métier
```

### Exemples de tests critiques

```typescript
// RequirementTable : filtres fonctionnels
it('filters requirements by type when filter is applied', () => {
  render(<RequirementTable requirements={mockRequirements} />)
  fireEvent.click(screen.getByText('Sécurité'))
  expect(screen.getAllByRole('row')).toHaveLength(3)  // header + 2 results
})

// DocumentUpload : erreur de format
it('shows error when unsupported file format is uploaded', async () => {
  render(<DocumentUpload projectId="123" />)
  const file = new File(['content'], 'test.exe', { type: 'application/x-msdownload' })
  await userEvent.upload(screen.getByLabelText('upload'), file)
  expect(screen.getByText(/Format non supporté/)).toBeInTheDocument()
})

// JobProgress : affiche la progression
it('updates progress bar as job progresses', async () => {
  const { rerender } = render(<JobProgress jobId="job-1" initialProgress={0} />)
  rerender(<JobProgress jobId="job-1" initialProgress={65} />)
  expect(screen.getByRole('progressbar')).toHaveAttribute('aria-valuenow', '65')
})
```

## Tests E2E (Playwright — Sprint 7)

```typescript
// tests/e2e/demo-flow.spec.ts
test('full demo flow: upload spec + scan repo + view conflict', async ({ page }) => {
  await page.goto('http://localhost:3000')

  // Créer un projet
  await page.click('text=Nouveau projet')
  await page.fill('[name=name]', 'TestProject')
  await page.click('text=Créer')

  // Uploader un document
  await page.setInputFiles('input[type=file]', 'tests/fixtures/demoshop/spec_checkout.md')
  await page.waitForSelector('text=Extraction terminée', { timeout: 60000 })

  // Vérifier les exigences extraites
  await page.click('text=Exigences')
  await expect(page.locator('table tbody tr')).toHaveCount(6)

  // Vérifier le conflit détecté
  await page.click('text=Impact')
  await expect(page.locator('[data-testid=conflict-alert]')).toBeVisible()
})
```
